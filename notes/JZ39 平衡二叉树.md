# JZ39 平衡二叉树
## 题目描述

> 输入一棵二叉树，判断该二叉树是否是平衡二叉树。

> 在这里，我们只需要考虑其平衡性，不需要考虑其是不是排序二叉树

## 解题思路
经典递归解法十分简洁。注意此题中的树根节点深度为1。

## 参考代码（C++）
```C++
class Solution {
public:
    int TreeDepth(TreeNode* pRoot){
        if (!pRoot) return 0;   // 若不含根节点则改为-1
        else return max(TreeDepth(pRoot->left), TreeDepth(pRoot->right))+1;
    }
};
```
