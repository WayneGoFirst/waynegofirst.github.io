# JZ38 二叉树的深度
## 题目描述

> 输入一棵二叉树，求该树的深度。从根结点到叶结点依次经过的结点（含根、叶结点）形成树的一条路径，最长路径的长度为树的深度。

## 解题思路
经典递归解法十分简洁

## 参考代码（C++）
    class Solution {
public:
    int TreeDepth(TreeNode* pRoot){
        if (!pRoot) return 0;
        else return max(TreeDepth(pRoot->left), TreeDepth(pRoot->right))+1;
    }
};
